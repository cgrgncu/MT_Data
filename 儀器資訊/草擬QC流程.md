# 草擬QC流程
+ 作者: HsiupoYeh

### LRN為例
+ 布置好測站，寫好紀錄紙，就立刻在NAS上建立對應資料夾，標註整理中，紀錄紙先拍照上傳到NAS中。
+ 收好測站資料後，長度一個月內的資料就進行壓縮，壓為zip。有好的網路的時候上傳。
+ 現場快速做幾個基本的資料處理:
  + 開啟一個專案叫做QC_LRN，匯入原始資料及CAL檔案。
  + 查看紀錄紙是否需要旋轉，展距長度是否正確。
  + 看一下有幾天的資料。
