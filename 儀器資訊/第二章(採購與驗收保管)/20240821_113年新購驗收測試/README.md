## 簡易保養方式
+ 主機及其配件、磁棒、磁棒纜線均為軍規防水產品，只要保存在室溫下乾燥的地方即可。搬運時拆卸纜線，小心不要碰撞或拉扯扭曲連接頭。
  + 若有必要可由專業人員進行檢測，檢測方式通常是在室外進行校正。
+ 電極為耗材，依照不同使用方式與頻率，開箱使用後壽命約一年至兩年。短時間保存時不必刻意加鹽水，只要保護好底座避免破損即可。若兩周以上沒有使用要在使用前將底部浸泡鹽水12小時。長時間保存原廠有建議，但對保管人來說不是很友善，個人建議單獨收好放鹽水海綿就好。
  + 若有必要可由專業人員進行檢測，檢測方式需要按照保養手冊之建議在室內建立實驗環境。
